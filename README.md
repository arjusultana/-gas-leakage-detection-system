# Gas Leakage Detection System

## Overview
This project detects LPG gas leakage using MQ2 gas sensor and alerts users using buzzer, LCD display and SMS notification.

## Features
- Gas leakage detection
- Buzzer alert
- LCD display warning
- SMS notification using GSM module

## Components Used
- Arduino UNO
- MQ2 Gas Sensor
- GSM Module
- LCD Display
- Buzzer
- Jumper Wires

## Technologies
- Arduino IDE
- Embedded C

## Future Improvements
- IoT integration
- Mobile app monitoring
- Automatic gas valve shutdown
